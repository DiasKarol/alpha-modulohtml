html e suas tags

Para desenvolver um documento html, basicamente precisamos de um editor de texto, como o Bloco de Notas, Visual Studio Code, JsFiddle, e muitos outros. Em seguida, precisamos saber que o documento deve ser salvo com a extensão .html, dessa forma, conseguimos abrir ele no navegador. Outra coisa importante a se saber, são as tags.


  1. Tags importantes

    Estas são as tags mais importantes para que os browsers façam a interpretação correta do documento:

    • <html> </html>: Esta tag é elemento basica da estruta a ser montada. Desta forma, Ela contera todas as tags que serão utilizadas, desta forma, ela deve estar no inicio e no fim do documento.

    • <head> </head>: Utilizada para delimitar o cabeçalho da página (utilizando-a no inicio e no fim das informações que serão destinadas a área em questão), Nem tudo que contem dentro dessa tag fica visível para o usuário final como, por exemplo, as folhas de estilo.

    • <meta charset=“utf-8” />: O "meta charset" informa ao browser qual é o conjunto de caracters utilizados naquele documento. Já o "UTF-8" contém todos os caracteres dos padrões Unicode e ASCII, sendo, portanto, o mais utilizado em páginas web

    • <title> </title>: Define o nome da página. Aquele nome que fica la na aba/janela do navegador.

      Exemplo: <title>Título da Página</title>

    • <body> </body>: É todo o conteudo visivel na página, tudo o que fica abaixo da aba/janela de navegação. Esta tag delimita o espaço e deve ter uma abertura e um fechamento, englobando as demais tags utilizadas no corpo do documento.


  2. Títulos

    Títulos são geralment utilizados para identificar a pagina, artigos e/ou seçoes. Dessa forma, no htmal temos 6 niveis para realizar esta identificação e elas são representados pelas tags <h1> </h1>, <h2> </h2>, <h3> </h3>, <h4> </h4>, <h5> </h5> e <h6> </h6>. Quanto menor o número, maior será a sua relevância.

    Para utilizar uma dessas tags, basta abrir a tag, acrescentar o conteúdo e fechar a tag.


  3. Paragrafos/Textos

    Para incluir textos ao documento, a tag mais utilizadas é a tag <p> </p>, neste caso, envolvemos o texto pela tag. É importante resoltar que esta tag se comporta como paragrafo, ou seja, cada paragrafo do documento deve ser envolto por uma tag. <p> </p>

    Exemplo:

    <p> Lorem ipsum dolor sit amet consectetur adipisicing elit. Repudiandae, corporis in rerum earum facere dicta id numquam, officia et, accusantium voluptates? Illum sequi maiores quam, impedit atque iure nulla necessitatibus. </p>

    <p> Lorem ipsum dolor sit amet consectetur, adipisicing elit. Tempora obcaecati nulla enim numquam dolorum? Facilis unde, pariatur, excepturi cupiditate optio eius corrupti ipsam est eveniet quibusdam velit corporis, dolorem adipisci. </p>


  4. Quebra de linha

    A tag <br /> é utilizada para quebra de linha. Muito utilizada quando se é necessario realizar uma significante divisão de linhas.

    Como, por exemplo, um poema. onde utilizariamos duas tags "p" e entre elas uma tag "br" para distanciar os paragrafos do poema.


  5. Link

    Para atribuir um link ao documento, pasta utilizar a tag <a> </a> junto do atributo href="endereçode.acesso". pode user utilizado para links externos e internos, basta por o endereço correto e completo. Entre a abertura e o fechamento da tag, adicione o texto que servira como link.

    Esta tag também pode ser utilizada para acessar um elemento na mesma página, basta utilizar o "id" no elemento a ser acessado e no "href" utilizamos "#nomedaid".

    Exemplo: <a href="endereçode.acesso"> texto de acesso </a>


  6. Imagem

    Caso queira adicionar uma imagem ao documento html, utilize a tag <img /> que recebe o endereço da imagem por meio do atributo src="imagem.png"

    outro atributo utilizado é o alt="texto alternativo" que irá descrever a imagem caso haja algum problema e o browser não consiga carregar a mesma.

    também utilizamor o atributo title="mensagem" que irá mostrar o texto atribuído a ela, no momento em que o usuario passar o mouse por cima da imagem.

    Exemplo: <img src="imagem.png" alt="texto alternativo" title="mensagem" />


  7. Listas

    Listas são elementos chaves quando precisamos organizar ou ordenar um itens e textos. No html, temos a ocpção de utilizar listas ordenadas(utilizando a tag <ol> </ol>) e não ordenadas (utilizando a tag <ul> </ul>).

    As tags informadas a cima, são utilizadas junto a tag <li> </li> que significa Item da Lista, ou seja, dentro de uma tag (ol ou ul) utilizamos uma <li> </li> para destacar com as marcações correspondentes da lista, os itens informados.

    Exemplo:

    <ul>

    <li> "texto" </li>

    <li> "texto" </li>

    <li> "texto" </li>

    </ul>
