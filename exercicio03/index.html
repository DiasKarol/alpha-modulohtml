Formatando o html

Ao formatar um texto html, devemos prestar atenção em todas as tags e não deixar de fechar nenhuma, pois iremos utilizar uma dentro da outra. Dessa forma, se esquecermos de fechar alguma tag, não iremos ter o resultado desejado. Além disso, podemos adicionar cor no texto, iremos acrescentar links ao textos.

Para saber sobre tags básicas, clique aqui!


Formatação de texto

Para incluir um palavra em negrito, italico, deixar a letra como formatação de computador, entre outros, utilizamos as segunties tags:

Tag Comportamento Exemplo Para usar
<b> </b> O texto ficara em negrito Texto negrito Texto em <b>negrito</b>
<i> </i> O texto ficara em italico Texto italico Texto em <i>italico</i>
<strong> </strong> O texto ficara destacado Texto destacado Texto <strong>Forte</strong>
<em> </em> O texto ficara enfatizado Texto enfatizado Texto em <em>enfatizado</em>
<sup> </sup> O texto ficara para cima Texto superior Texto em <sup>superior</sup>
<sub> </sub> O texto ficara para baixo Texto inferior Texto em <sub>inferior</sub>
<del> </del> O texto ficara taxado Texto taxado Texto em <del>taxado</del>
<code> </code> O texto ficara como código de computador código de computador código de <code>computador</code>
<pre> </pre> O texto ficara pré-formatado
